# Jackabrew Portfolio Radar

Streamlit dashboard + helper CSVs for tracking equity/crypto holdings, allocations, and watchlist reminders.

## Structure

- `dashboard/` – Streamlit app (entry point `app.py`)
- `holdings.csv` – Current positions (Market, Ticker, Name, Shares)
- `watchlist.csv` – Optional watchlist + target prices

See `dashboard/README.md` for run/deploy instructions.
